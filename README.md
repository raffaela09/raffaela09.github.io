# Topo engenharia - Site institucional

Site institucional desenvolvido como projeto freelancer para apresentação
da empresa e seus serviços.

## 📌 Sobre o projeto
Projeto desenvolvido com foco em:
- layout moderno
- responsividade
- performance
- facilidade de manutenção de conteúdo

## 🚀 Tecnologias utilizadas
- HTML5
- CSS3
- JavaScript
- Conteúdo dinâmico via JSON

## 🎨 Funcionalidades
- Página inicial institucional
- Layout responsivo (mobile e desktop)
- Conteúdo centralizado em arquivo configurável
- Estrutura otimizada para manutenção
- Integração com botões de contato

## 🛠 Como executar o projeto

1. Baixar ou clonar o repositório
2. Abrir em um servidor local (ex: Live Server)
3. Acessar index.html

## ✏️ Como editar conteúdo
Os textos, cores e imagens estão centralizados no arquivo:

content.json

Altere apenas os valores mantendo a estrutura do arquivo.
